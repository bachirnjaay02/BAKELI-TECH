# Bakeli Tech

Ce projet est un site vitrine pour Bakeli, une école de formation professionnelle dans les nouvelles technologies créée par Volkeno. Le site présente les formations, les coachs, les événements, le blog, les bourses et permet de contacter ou de suivre Bakeli sur les réseaux sociaux.

## Fonctionnalités principales
- Navigation claire et moderne
- Section d'accueil avec présentation de l'école
- Liste des formations et des coachs
- Présentation des événements à venir
- Blog et bourses
- Footer avec liens vers les réseaux sociaux (Instagram, Facebook, LinkedIn, X)
- Responsive design (adapté aux mobiles et ordinateurs)
- Header fixe lors du scroll
- Animation sur les cards d'événements

## Technologies utilisées
- HTML5
- CSS3

## Démarrage rapide
1. Clonez le dépôt :
   ```
   git clone <url-du-depot>
   ```
2. Ouvrez le dossier dans votre éditeur (VS Code recommandé).
3. Ouvrez `Bakeli tech.html` dans votre navigateur pour visualiser le site.
